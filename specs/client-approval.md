## 1. Стек технологий

| Что | Решение | Аналог из ТЗ |
|-----|---------|-------------|
| CMS | Next.js + Supabase (кастомная админка) | WordPress |
| Мультиязычность | next-intl (встроено в код) | WPML |
| Конструктор | Не нужен — админка с формами | Elementor |
| YouTube | Нативные iframe (lazy-load) | Smash Balloon |
| RSS-парсинг | Кастомный парсер (API route + cron) | WP RSS Aggregator |
| SEO | Встроено в Next.js (meta, OG, sitemap, hreflang) | Yoast SEO |
| Кэширование | Vercel Edge + ISR | WP Rocket |
| Безопасность | Supabase RLS + middleware + CSP headers | Wordfence |
| Бэкапы | Supabase автобэкапы (daily, 30 дней) | UpdraftPlus |
| GDPR | Кастомный cookie banner + consent checkboxes | Complianz |
| Хостинг | Vercel (global CDN, SSL, preview deploys) | VPS/Kinsta |

**Преимущества перед WordPress:**
- Скорость: статическая генерация + edge CDN → PageSpeed 95+
- Безопасность: нет PHP, нет плагинов, нет уязвимостей WP
- Деплой: каждый пуш = preview URL, откат за 1 клик
- Масштабируемость: Vercel бесплатно выдерживает любой трафик
- Стоимость: хостинг $0/мес (Vercel free tier) vs $30–50/мес VPS

## 6. Процесс работы (информирование заказчика)

1. **Wireframes** — каркас каждой страницы в Figma → утверждение заказчиком
2. **Дизайн** — компоненты на реальных данных → утверждение
3. **Разработка** — по одной странице, каждая с preview-ссылкой
4. **Тестирование** — автоматические + ручные тесты
5. **Запуск** — перенос на домен tihikos.com

**Заказчик получает:**
- Figma-ссылку для комментирования wireframes
- Preview URL после каждой готовой страницы
- Доступ к админке после Фазы 1
